# Changelog

All notable changes to the XYBERSEAN Code of Conduct are recorded here.

## [1.0] — 2026-04-28

### Added
- Initial publication.
- 12 sections: preamble, core principles, operator categories, system
  access responsibilities, data handling, legal compliance, AI agent
  duties, hacker ethics canon, forbidden actions, incident response,
  acceptance, versioning.
- SSH pre-auth banner (`coc-banner.txt`).
- Interactive shell hook (`coc-motd.sh`) for sessions that bypass SSH.
- Idempotent fleet installer (`install.sh`) supporting macOS and Linux.
